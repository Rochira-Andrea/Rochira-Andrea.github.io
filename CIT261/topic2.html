More about object Inheritance

In JavaScript is possible to create an object from one or more existing objects.

The new object inherits all the properties of the source object(s).

Let's start from two parents object:

var Elf = { race:"Sindarin Elf", provenance:"Woodland Realm" };
var Weapon = { bow:"bow", dagger:"daggers" };

The forms below show different ways to accomplish the task described.

In the first form a new object is created from Elf (and it has additional properties).

In the second form is performed the same operation but using the spread operator.

In the third form a new object is created by combining Elf and Weapon.
